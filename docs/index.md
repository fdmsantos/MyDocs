## Commands

* `py -m mkdocs new [dir-name]` - Create a new project.
* `py -m mkdocs serve` - Start the live-reloading docs server.
* `py -m mkdocs build` - Build the documentation site.
* `py -m mkdocs help` - Print this help message.
